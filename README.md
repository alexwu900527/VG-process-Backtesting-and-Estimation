# VG過程的風險值估計與實證回測

風險管理對於金融機構來說是非常重要的課題，根據以往的幾次重大的金融危機，如1997年的亞洲金融風暴、2008年的次貸風暴或2020年新冠疫情引發的金融恐慌，這些危機多因為不當的風險管理，如寬鬆監管下的信貸擴張、資產泡沫及高槓桿導致，因此也大大地增加了風險管理的重要性。而變異數伽馬過程（Variance Gamma process, VG process）作為非常具有金融直觀且極具應用價值的隨機過程，在選擇權估價方面已有相當多研究去探討，應用在市場風險管理上的研究卻不多，因此本研究專注於呈現VG過程在美股指數的風險模型表現，並提供一套完整可實行的估計方法。

在市場風險的監理架構中，市場風險資本計提（Market Risk Capital Charge）為銀行計算出來需要應對市場波動而提取的準備金，確保銀行能承受金融市場的激烈波動如股災。根據新巴塞爾資本協定（Basel II）和巴塞爾協定III（Basel III），風險值（Value-at-Risk, VaR）與預期損失（Expected Shortfall, ES, 亦稱 CVaR）為核心風險衡量的指標，VaR表示在某信心水準下可能遭遇的最大損失，ES則是當損失已超過VaR值時的平均損失。舉例來說：如果99%的信心水準下最多會損失100，且當損失超過100時平均會損失135，則99% VaR為100、99% ES為135。相較於VaR，ES能夠衡量在極端損失情境下的平均損失水準，理論上更符合風險管理對尾端風險與系統性風險的監理需求。不過在風險模型驗證與回測（Backtest）方面，銀行主要計算VaR並進行回測，作為監理門檻的判斷依據，也是本研究所採取的回測驗證依據。原因是ES並不存在「超越事件」，使其無法對ES透過傳統的違約次數或違規比率進行直接回測，而是結合多重驗證機制以確保模型的風險刻畫能力，因此本研究把ES當成解釋模型的參考值，不把它跑入回測統計檢定。

金融機構如銀行於壓力狀態下，至少要兩個星期，也就是10個交易日的準備時間，才能把部位清掉或避險，故Basel II與Basel III都要求VaR或ES的天數至少為10天，而Basel 2要求99% VaR、Basel 3要求的97.5% ES來作為市場風險資本計提的因子。不過以往的隨機過程風險模型之回測實證研究多直接關注在單日的VaR，較少討論多日VaR的情況，本研究專注於10日的報酬尺度，以NASDAQ、S&P500與QQQ為研究對象，比較VG過程與幾何布朗運動（Geometric Brownian Motion, GBM）、加權歷史模擬法（Weighted Historical Simulation, WHS）、GARCH模型對實證報酬分配的刻畫能力。
